# Игра «Угадай число» на чистом JavaScript

Консольная браузерная игра, в которой компьютер загадывает случайное число от 1 до 100, а у игрока есть 7 попыток, чтобы его отгадать.

### 🎯 Цель проекта
Отработка базовых конструкций JavaScript: циклов `do...while`, операторов ветвления `if/else`, генерации случайных чисел и строгой валидации пользовательского ввода.

### 🚀 Ключевые особенности реализации
* **Надежная валидация (Защита от дурака):** Программа корректно обрабатывает пустые строки, ввод букв вместо цифр (через `Number.isNaN`), а также безопасное прерывание игры при нажатии кнопки «Отмена».
* **Динамические подсказки:** После каждого неверного ответа игра сообщает, больше или меньше загаданное число.
* **Ограничение по попыткам:** Реализован счетчик декремента попыток, завершающий игру при достижении нуля.
